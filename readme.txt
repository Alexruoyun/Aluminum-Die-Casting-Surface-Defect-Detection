This dataset contains the pre-processed images of die-case gas meter lids. 

The three folders represents the subsets for training, validation, and test, respectively.

Inside each folder, there are three categories: 0 represents flawless, 1 represents crack, and 2 represents cold flow.

Please refer to the following article for details about the dataset. Please also cite the article if you use this dataset.
